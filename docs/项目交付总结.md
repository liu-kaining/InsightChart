## 🚀 部署就绪

### 部署前准备
1. **环境依赖**: Docker + Docker Compose
2. **API密钥**: 配置Qwen或DeepSeek API密钥
3. **访问口令**: 设置安全的访问密码

### 一键部署
```bash
# 1. 配置环境变量
cp .env.example .env
vim .env  # 填入API密钥

# 2. 一键部署
./deploy.sh

# 3. 访问应用
open http://localhost:5000
```
